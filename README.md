## Installation

1. Clone the repository:
```bash
git clone https://github.com/lildripboi123-jpg/purk-sss.git
cd purk-sss
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The site will open at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Customization

### Update Brand Name
Replace "PURK" and "We The Ones" throughout the site.

### Change Colors
Edit the color variables in `src/index.css` and component CSS files:
- Primary color: `#d946ef`
- Background: `#0a0a0a`

### Add Products
Update the products array in `src/pages/Shop.js` and `src/pages/Home.js`

### Connect Social Media
Update social media links in `src/components/Footer.js` and `src/pages/Contact.js`
